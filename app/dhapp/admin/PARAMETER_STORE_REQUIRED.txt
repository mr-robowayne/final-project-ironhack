Recommended in AWS SSM Parameter Store (non-secret)
Date: 2026-03-11

frontend (public build/runtime config)
- REACT_APP_API_BASE
- REACT_APP_API_URL
- REACT_APP_DEFAULT_TENANT
- REACT_APP_LT_API_URL
- REACT_APP_CHAT_NOTIFICATION_PREVIEW
- REACT_APP_BACKEND_PORT (dev only)
- REACT_APP_DEV_TENANT_ID (dev only)
- REACT_APP_DEV_USER_ID (dev only)

backend/login/billing/ai (examples)
- APP_ENV, NODE_ENV
- DB_HOST, DB_PORT, DB_NAME, DB_USER, DB_SSLMODE
- CORS_ORIGIN, CORS_ALLOWLIST
- DEFAULT_TENANT_ID
- SESSION_COOKIE_SECURE, SESSION_COOKIE_SAMESITE
- AI_SERVICE_URL, BILLING_SERVICE_URL
- REDIS_URL
- S3_REGION, S3_PATIENT_FILES_BUCKET, S3_AI_SOURCE_BUCKET

Rule
- Keep sensitive values (passwords/tokens/keys) in Secrets Manager, not Parameter Store String.
