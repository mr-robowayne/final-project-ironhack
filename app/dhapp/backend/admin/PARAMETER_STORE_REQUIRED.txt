Rule
- Put only non-sensitive values in Parameter Store.
- Sensitive values (JWT_SECRET, DB_PASSWORD, OPENAI_API_KEY, *_TOKEN) go to Secrets Manager.
- No AWS_ACCESS_KEY_ID or AWS_SECRET_ACCESS_KEY — EC2 instance role provides S3/KMS access.
